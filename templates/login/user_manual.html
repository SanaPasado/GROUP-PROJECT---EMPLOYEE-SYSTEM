Employee Management System

Quick User Guide

1. Login & Security
  • Enter username and password.
  • If IP differs from last login → system asks for Google Authenticator code (QR scanned during setup).
  • Code changes every 30 seconds for extra security.
2. Roles
  • Employee: View/update profile, time in/out, request leave.
  • Admin/HR: Manage employees, view all attendance, create accounts, generate reports.
3. Employee Actions
  • My Profile → View/update personal info.
  • My Attendance → Time In (green) / Time Out (red).
  • Recent Records → See last 10 logs.
4. Admin Actions
  • Employees → Add/edit employee accounts (admins or authorized employees can create accounts).
  • Attendance List → Check all employees’ time in/out.
  • Reports → Generate/export attendance or payroll reports.
5. Tips
  • Always log out after use.
  • Keep contact info updated.
  • Report login issues to admin.
